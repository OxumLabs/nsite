Curious about the syntax? Neit keeps it simple and elegant, so you spend less time wrestling with errors and more time building amazing things. Here’s a sneak peek to show you just how natural it feels:

print Hello world # Prints "Hello world!"
may name = "joy" # Creates a variable 'name' and sets it to "joy"
print {name} # Prints the value of 'name' (which is "joy")
println {name} # Prints the value of 'name' on a new line
may age = takein() # Asks the user for input, defaulting to a string

And for those who like their code with a bit of personality, here’s a quick example:

println "Hello user! What’s your name?"
print ">"
may name = takein() # Takes input from the user
println "Hello {name}! Welcome to Neit!" # Greets the user by name
