Neit Development Tools

Here are some amazing tools that make working with Neit even more powerful and fun! These tools help with compilation, development, and integration with popular editors.


<~

Available Tools:

  • Nulibc - A C file that Neit links during C code compilation. It's a standard library in development for Neit, providing useful features to enhance your coding experience.

  • Nide - A Neit IDE for easier development (Coming Soon!). It will feature code completion, syntax highlighting, code refactoring, debugging and more. It will be a fully fledged IDE for Neit which will make it easier to develop applications with Neit.

  • Neit VSCode Extension - A VSCode extension for Neit to enhance your coding experience with syntax highlighting, code completion, debugging and more. It adds Neit support to VSCode so you can write Neit code in a more comfortable environment.
